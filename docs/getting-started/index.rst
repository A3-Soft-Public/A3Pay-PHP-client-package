###############
Getting started
###############

This package provide an easy way to integrate FiskalPay payment gateway using OOP structure.

You can make payment by install this library, provide necessary data, and just make request.

Request will return a response with redirect url. After processing the payment, the payment gateway will redirect to given link, and notify you by provided web hook
