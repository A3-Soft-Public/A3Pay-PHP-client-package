Usage

We will use code snippets from tests to demonstrate how to use this library.

The core of this package is PaymentGatewayRequester service.

All classes from namespace Helper are just data models required to make API request.

Constructing service.

If we want to construct PaymentGatewayRequester service, we need url link and access token

example.php 
<?php

    $paymentGatewayRequester = new PaymentGatewayRequester('https://api_url_from_registration_request', 'token_from_registration_request');

OK, if we have PaymentGatewayRequester service, we need to make request. Requests are made using PaymentGatewayRequester::makeRequest() method.

This method require array of key - value pairs, or an object implementing PaymentGatewayRequest interface.

We know two types of request:

  • Payment request
  • Info request
